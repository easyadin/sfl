Intended Use

Stress Free Learning is a playful testing platform for dyslexia in school children

By assembling Thymio robots, the children spell different words. An administrator supervises the process and takes measurements such as completion time, error rates etc.

On that basis, an indication can be made if the child might be suffering from dyslexia.

Background Algorithm

In English, there are no fixed spelling rules. Therefore, there is no sure way to tell which letters are acceptable next to each other

In English, there are no fixed spelling rules. Therefore, there is no sure way to tell which letters are acceptable next to each other.

However, there are tendencies for certain letters to occur next to each other. To give a simple example, Q will be followed by U in almost all cases. The current implementation makes use of this fact to predict how likely it is for two letters to be next to each other.
